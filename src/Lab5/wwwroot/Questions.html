i. List and describe 3 types of static content that can be used in MVC Core

Three different types of static content are Javascript scripts, Images, and HTML pages.

ii. Where does static content belong in an MVC Core application?

Static content belongs in the wwwroot folder of the project.

iii. Explain how to link to a static page from a controller and/or view.

To link a static page from a controller or a view you would have to use a link with an href field (ex. ~/Example.html). In a controller you would have to use the Redirect function with the same sort of url as href.
